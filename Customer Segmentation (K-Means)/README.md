# Customer Segmentation using K-Means

## Objective

Segment customers based on spending behavior using K-Means clustering.

## Dataset

Mall Customers Dataset

## Techniques Used

-   K-Means Clustering
-   PCA

## Steps

-   EDA
-   Feature Scaling
-   Elbow Method
-   Clustering
-   Visualization

## Results

5 customer segments identified based on income and spending patterns.

## Business Impact

Helps businesses target the right customers with the right strategy.
